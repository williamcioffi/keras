fridgebot

fridgebot keeps track of the fridge temperature so reagents don't spoil. also it tweets sometimes.

I put fridgebot together using the following components, plus some odds and ends.

  • arduino uno
  • MCP9808 temperature sensor (adafruit)
  • photoresistor
  • sd card shield (seeduino)
  • FONA mini cellular GSM board (adafruit)
I used an old sd card shield from seeduino that I had lying around, but there are plenty of other arduino compatable sd card interfaces that should work well. Both the temperature sensor and the cellular unit came from adafruit and have complementary step by step tutorials and custom libraries that made everything very easy.
